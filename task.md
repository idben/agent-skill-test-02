# 任務：建立數字計算網頁

## 📋 需求分析
建立一個網頁，讓使用者輸入兩個數字後，彈出訊息告知計算結果。

## 🔧 實作細節

### 檔案結構
- `index.html` - 單一 HTML 檔案（包含 CSS 和 JavaScript）

### 功能規劃
1. 兩個數字輸入欄位
2. 計算按鈕
3. 使用 HTML `<dialog>` 元素顯示結果（取代傳統 alert，提供更好的使用者體驗）

### 技術規格
- 使用 `querySelector` 語法
- 小駝峰變數命名
- 兩空白縮排
- 純 HTML/CSS/JavaScript，無框架

## ✅ 已確認事項
1. 計算類型：僅加法
2. 使用 `<dialog>` 取代 `alert()`：同意
